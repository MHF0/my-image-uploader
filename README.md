## ğŸ› ï¸ Installation & Setup

### Prerequisites
- Node.js (v22+)
- Go (v1.24+)
- Docker (optional)

### Frontend Setup
```bash
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### Backend Setup
```bash
# Navigate to server directory
cd server

# Run the Go server
go run main.go
```

### Docker Setup (Optional)
```bash
# Build and run with Docker Compose
docker-compose up -d
```
